Cleaners Hub - the harmony network 

App Specification

App Overview

A simple, intuitive service booking app where users can find and hire cleaners, cooks, caregivers, and other home service professionals. Clean sky-blue and white design with smooth animations.

USER TYPES & ACCESS

1. Regular Users (Clients)

People who need services for their homes, offices, or events.

2. Service Providers

· Individual cleaners/cooks/caregivers
· Cleaning teams/companies
· Other service professionals

3. Admin Users

Platform managers who oversee everything



COLOR SCHEME & DESIGN

· Main Color: Sky Blue (#87CEEB)
· Background: White
· Accent Color: Soft Gray (#F5F5F5) for section backgrounds
· Button Color: Sky Blue with white text
· Special Effects:
  · Gentle water ripple when tapping buttons
  · Small sparkle animation when completing actions
  · Smooth slide transitions between screens

SPECIAL FEATURES FOR SMOOTH EXPERIENCE

1. Smart Search

· Search by service, location, or provider name
· "Popular near you" suggestions
· Recent searches saved

2. Quick Rebooking

· From past bookings, one-tap to book same service again
· "Usual time" remembered for each provider

3. Real-Time Tracking

· For booked services: see provider's location (with permission)
· Estimated arrival time updates
· "On the way" notification

4. Photo Verification

· Providers can upload ID for verification
· Background check status shown
· Trust badges on profiles

5. Easy Payments

· Save multiple payment methods
· Split payment option (pay half now, half later)
· Receipts automatically saved

6. Smart Notifications

· Booking reminders 24 hours before
· Provider on the way alert
· Payment receipts
· Review reminders after service

7. Emergency Mode

· Red "Emergency" button on home screen
· Connects to nearest available provider
· Higher priority, faster response

8. Family Sharing

· Add family members to your account
· Share booking history
· Set permissions (who can book services)

9. Multi-Language Support

· Switch between languages in settings
· App available in English, Spanish, French
· More languages added based on location

10. Offline Mode

· View past bookings offline
· Draft messages saved
· Reconnect when back online



BUTTON CONNECTIONS SUMMARY

Client App Navigation:

1. Home Screen → Find Services / Book Now / My Bookings / Messages
2. Find Services → Provider Profile → Book Now → Booking Screen → Confirmation
3. Book Now → Quick Service Selection → Booking Screen → Confirmation
4. My Bookings → Booking Details → Reschedule/Review/Cancel
5. Messages → Individual Chat → Call/Share Location
6. Profile → All Settings & Account Management

Provider App Navigation:

1. Dashboard → My Schedule / New Requests / My Clients / My Earnings
2. My Schedule → View/Edit Availability
3. New Requests → Accept/Decline Bookings
4. Profile → Edit All Business Details

Admin App Navigation:

1. Dashboard → All Management Sections
2. Manage Users → View/Suspend/Message Users
3. Manage Bookings → View/Modify/Resolve Bookings
4. App Settings → Configure Platform Rules



ANIMATIONS & TRANSITIONS

1. Screen Transitions: Slide left/right (like turning pages)
2. Button Press: Gentle water ripple effect from tap point
3. Loading: Rotating circle with sparkle dots
4. Success Actions: Small sparkle burst with checkmark
5. Error States: Gentle shake with red highlight
6. Scroll Effects: Fade in content as you scroll
7. Refresh: Pull down with water wave animation



TECHNICAL REQUIREMENTS FOR AI TO BUILD

Frontend:

· React Native or Flutter for cross-platform
· Sky blue (#87CEEB) as primary color theme
· Simple, large buttons (minimum 44px touch target)
· Clear typography (San-serif fonts)
· Icon system for navigation

Backend:

· User authentication system (3 types)
· Service provider verification pipeline
· Booking management system
· Real-time chat system
· Payment processing integration
· Review and rating system
· Notification system (push/email/SMS)

Database:

· User profiles (clients, providers, admins)
· Service listings and categories
· Booking records
· Messages and conversations
· Payment transactions
· Reviews and ratings

APIs Needed:

· Map services for location/distance
· Payment gateway (Stripe/PayPal)
· SMS/Email service for notifications
· File storage for photos
· Analytics for usage tracking

Security:

· Secure login with password hashing
· Payment data encryption
· Background checks for providers
· Privacy controls for users
· Admin access logging



USER TESTING CHECKLIST

· Can a new user book a service in under 3 minutes?
· Can providers easily manage their schedule?
· Are all buttons clearly labeled and easy to tap?
· Is the payment process simple and secure?
· Does the app work on both old and new phones?
· Are error messages helpful and clear?
· Can users contact support easily?
